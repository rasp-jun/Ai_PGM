# 📦 Python Container(컨테이너)

## 컨테이너(Container)란?

컨테이너(Container)는 **여러 개의 데이터를 하나의 변수에 저장할 수 있는 자료형**을 말한다.

파이썬의 대표적인 컨테이너는 다음과 같다.

* List (리스트)
* Tuple (튜플)
* Dictionary (딕셔너리)
* Set (집합)

---

# 1. List (리스트)

리스트는 **순서가 있으며(Ordered), 수정 가능한(Mutable)** 자료형이다.

### 특징

* 순서(O)
* 중복 허용(O)
* 수정 가능(O)

### 생성

```python
numbers = [1, 2, 3]
```

### 주요 메서드

| 메서드       | 설명             |
| --------- | -------------- |
| append()  | 요소 추가          |
| insert()  | 원하는 위치에 삽입     |
| remove()  | 값 삭제           |
| pop()     | 마지막 요소 삭제 및 반환 |
| sort()    | 정렬             |
| reverse() | 역순 정렬          |
| clear()   | 모든 요소 삭제       |

---

# 2. Tuple (튜플)

튜플은 **순서가 있지만 수정이 불가능한(Immutable)** 자료형이다.

### 특징

* 순서(O)
* 중복 허용(O)
* 수정 불가(X)

### 생성

```python
point = (10, 20)
```

### 장점

* 속도가 빠름
* 데이터 변경 방지
* 함수의 반환값으로 많이 사용

---

# 3. Dictionary (딕셔너리)

딕셔너리는 **Key와 Value를 한 쌍으로 저장하는 자료형**이다.

### 특징

* Key : 중복 불가
* Value : 중복 가능
* 순서 유지(Python 3.7 이상)

### 생성

```python
student = {
    "name": "Kim",
    "age": 20
}
```

### 주요 메서드

| 메서드      | 설명            |
| -------- | ------------- |
| keys()   | 모든 Key 반환     |
| values() | 모든 Value 반환   |
| items()  | Key와 Value 반환 |
| get()    | Value 조회      |
| update() | 데이터 수정        |
| pop()    | 데이터 삭제        |

---

# 4. Set (집합)

집합(Set)은 **중복을 허용하지 않는 자료형**이다.

### 특징

* 순서(X)
* 중복(X)
* 수정 가능(O)

### 생성

```python
numbers = {1, 2, 3}
```

### 주요 연산

```python
A | B      # 합집합
A & B      # 교집합
A - B      # 차집합
```

### 주요 메서드

| 메서드       | 설명           |
| --------- | ------------ |
| add()     | 요소 추가        |
| update()  | 여러 요소 추가     |
| remove()  | 요소 삭제        |
| discard() | 요소 삭제(오류 없음) |
| clear()   | 모든 요소 삭제     |

---

# 컨테이너 비교

| 자료형        | 순서      | 중복              | 수정 |
| ---------- | ------- | --------------- | -- |
| List       | O       | O               | O  |
| Tuple      | O       | O               | X  |
| Dictionary | O(3.7+) | Key X / Value O | O  |
| Set        | X       | X               | O  |

---

# 언제 사용할까?

* **List** : 순서가 있는 데이터를 저장하고 수정해야 할 때
* **Tuple** : 변경되면 안 되는 데이터를 저장할 때
* **Dictionary** : Key를 이용해 빠르게 데이터를 찾을 때
* **Set** : 중복 제거 또는 집합 연산이 필요할 때

---

# 요약

| 자료형        | 대표 특징                  |
| ---------- | ---------------------- |
| List       | 가장 많이 사용하는 컨테이너, 수정 가능 |
| Tuple      | 읽기 전용 데이터 저장           |
| Dictionary | Key-Value 형태의 데이터 저장   |
| Set        | 중복 제거 및 집합 연산          |
